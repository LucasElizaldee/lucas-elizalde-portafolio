# Repositorio de competencias - Lucas Elizalde

Este ZIP contiene solamente los 4 apartados de competencias para subir a GitHub Pages:

- operacion/index.html
- mantenimiento/index.html
- diseno/index.html
- innovacion/index.html

No se agregó barra de navegación superior ni se modificó el diseño de los HTML originales.

Cada apartado tiene carpetas preparadas para multimedia:

- img/
- videos/
- docs/

## URLs esperadas en GitHub Pages

portafolio-competencias-html

https://TU_USUARIO.github.io/portafolio-competencias-html/operacion/
https://TU_USUARIO.github.io/portafolio-competencias-html/mantenimiento/
https://TU_USUARIO.github.io/portafolio-competencias-html/diseno/
https://TU_USUARIO.github.io/portafolio-competencias-html/innovacion/

Luego esas URLs se incrustan en las páginas correspondientes de Google Sites.
