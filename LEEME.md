# Competencias - Portafolio Lucas Elizalde

Este repositorio contiene solo los 4 HTML de competencias para incrustar en Google Sites:

- `operacion/index.html`
- `mantenimiento/index.html`
- `diseno/index.html`
- `innovacion/index.html`

También contiene una carpeta compartida:

- `evidencias-media/`

Ahí se suben las fotos, videos y documentos de las 28 evidencias.

## URLs esperadas

Si el repositorio se llama `portafolio-competencias-html`, las URLs serán:

```text
https://TU_USUARIO.github.io/portafolio-competencias-html/operacion/
https://TU_USUARIO.github.io/portafolio-competencias-html/mantenimiento/
https://TU_USUARIO.github.io/portafolio-competencias-html/diseno/
https://TU_USUARIO.github.io/portafolio-competencias-html/innovacion/
```

Esas URLs se incrustan en las páginas correspondientes de Google Sites.

## Cómo subir fotos

Cada evidencia tiene una carpeta:

```text
evidencias-media/NOMBRE-DE-LA-EVIDENCIA/img/
```

Las imágenes deben llamarse:

```text
imagen-1.jpg
imagen-2.jpg
imagen-3.jpg
```

según la cantidad indicada en cada actividad.

## Cómo mostrar informe desde Google Drive

Dentro de cada evidencia hay un archivo:

```text
drive-link.txt
```

Pegá ahí el enlace completo del informe de Google Drive. Ejemplo:

```text
https://drive.google.com/file/d/ID_DEL_ARCHIVO/view?usp=sharing
```

El HTML usará ese enlace para mostrar el documento dentro de la página con el modo `/preview`.

## Cómo subir video local

Si querés usar video local, subilo como:

```text
evidencias-media/NOMBRE-DE-LA-EVIDENCIA/videos/demo.mp4
```
